# Games on Web 2026

***

## Casino Pipo

Casino Pipo est un prototype de casino en ligne développé pour le projet Games on Web 2026. Il s'agit d'une application web 3D (moteur Babylon.js) qui regroupe plusieurs jeux de casino (blackjack, poker, machines à sous) avec des composants d'IA.

**Technologies principales :** Babylon.js, Vite, JavaScript.

## Installation

Pré-requis : Node.js (recommandé >= 16) et npm.

- Installer les dépendances :

```bash
npm install
```

- Lancer le serveur :

```bash
npm run dev
```
